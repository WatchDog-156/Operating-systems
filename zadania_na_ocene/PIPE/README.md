Napisano "mini-shell" - program, który wczytuje polecenie z wiersza poleceń. Akceptuje on poniższe polecenia:

**run command** - uruchom wskazane polecenie (ścieżka względna lub bezwzględna)

**redirect command1 to command2** - uruchom dwa wskazane polecenia przekierowując wyjście comman1 do command2

**exit** - kończy działanie programu
